# 阶段16：目标期刊定向润色

## 目标
在不改变事实的前提下，使稿件更贴近目标期刊的风格、表达重心和读者预期。

## 这一阶段要做什么
1. 根据目标期刊调整标题风格
2. 调整摘要密度与重心
3. 调整引言中的 gap 表达
4. 调整讨论中的 novelty / significance 表达强度
5. 优化整体语言风格与节奏

## 输入
- Full manuscript
- Target journal
- Journal style notes

## 输出
- 目标期刊导向版本 manuscript
- 风格修改说明
- 仍需注意的风险点

## 注意事项
- 这一步只改表达与重心，不新增事实
- 不要为了迎合期刊而强行夸大结论
- 如果某些结论对该期刊来说过强，应主动收紧

## 完成标志
- [ ] 标题已贴近期刊风格
- [ ] 摘要已贴近期刊风格
- [ ] 全文表达重心已调整
- [ ] 风格性风险点已标记
