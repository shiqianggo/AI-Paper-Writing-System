# 阶段08：Discussion专项质控

## 目标
控制过度解释、模板化讨论和证据越界。

## 这一阶段要做什么
重点检查：
1. 是否把相关性写成因果
2. 是否有超出 Results 支持范围的推断
3. 是否过于模板化
4. 是否真正突出研究独特贡献
5. 是否需要补 limitations 或解释边界

## 输入
- Discussion 初稿
- Results
- Validated Results summary

## 输出
- Discussion 质控报告
- 修订后的 Discussion

## 高风险信号
- “This demonstrates that...”
- “This proves...”
- 从局部模式直接推到广泛机制
- 讨论段落换篇文章也能成立
- 讨论太顺，但太空

## 完成标志
- [ ] 过强措辞已收紧
- [ ] 过度解释已删除或降级
- [ ] 研究特异性增强
- [ ] limitations 已评估
- [ ] Discussion 基本稳定
