# 阶段09：写 Abstract / Title / Keywords

## 目标
在主体章节稳定后，从全文反向压缩生成摘要、标题和关键词。

## 这一阶段要做什么
1. 基于完整 manuscript 草稿生成 Abstract
2. 生成多个标题候选
3. 提炼关键词
4. 保持与正文一致

## 输入
- Full manuscript draft
- Journal style preference
- Title style preference

## 输出
- Abstract
- Title 候选
- Keywords

## 核心要求
- 摘要必须忠实于正文
- 标题要具体、简洁、可信
- 关键词要覆盖对象、概念、方法和过程
- 避免比正文更强的机制性表述

## 注意事项
- 这一步必须放在主体章节后面
- 不要过早写摘要
- 标题不要只追求“好看”，要先可信

## 完成标志
- [ ] Abstract 已完成
- [ ] 标题候选已生成
- [ ] Keywords 已确定
- [ ] 进入摘要与标题专项质控
