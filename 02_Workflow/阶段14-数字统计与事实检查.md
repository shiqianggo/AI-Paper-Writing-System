# 阶段14：数字统计与事实检查

## 目标
单独检查论文中的数字、样本量、统计结果和关键事实表述。

## 这一阶段要做什么
重点检查：
1. 是否有来源不清的数字
2. 样本量、基因数、代谢物数、百分比是否前后一致
3. 统计指标是否有名称、上下文和解释
4. 是否存在“good performance”但缺具体指标
5. 数字表达是否规范

## 输入
- Full manuscript
- Validated Results summary
- Key numbers table

## 输出
- 数字与统计问题清单
- 需要人工确认的项目列表
- 统一后的数字表述建议

## 为什么重要
AI 稿件最容易出现的高风险问题之一就是：
“文字很像真的，但数字口径混乱。”

## 注意事项
- 核心数字最好逐条核对
- Results、Abstract、Discussion 中的数字应一致
- 注意不要把 annotation 数、feature 数、detected 数混写

## 完成标志
- [ ] 数字来源已清楚
- [ ] 口径已统一
- [ ] 统计表述已规范
- [ ] 高风险数字已人工确认
