# 阶段11：去AI味与增强研究特异性

## 目标
降低 AI 模板感，增强作者主导感和研究专属性。

## 这一阶段要做什么
1. 重写过度平滑、过度模板化的句子
2. 收紧空泛表述
3. 增加对本研究系统和结果的贴合度
4. 保持事实不变，只优化表达方式

## 适用对象
- Introduction
- Discussion
- Abstract
- Results 中的段落收束句

## 输出
- 更像作者写的版本
- 研究特异性更强的版本

## 高风险信号
- 换个研究也能用的段落
- 很顺，但没有研究特征
- 宏大、抽象、像综述
- 结尾句过于“高级但空”

## 注意事项
- 不新增事实
- 不新增文献
- 不修改科学结论
- 重点提升“这篇文章专属感”

## 完成标志
- [ ] 模板化表达明显减少
- [ ] 作者主导感增强
- [ ] 研究系统特异性增强
- [ ] 关键章节完成语言收紧
