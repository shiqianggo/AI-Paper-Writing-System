# 阶段06：Introduction专项质控

## 目标
确保引言真正服务于本研究，而不是停留在模板化背景描述。

## 这一阶段要做什么
重点检查：
1. 背景是否太宽泛
2. gap 是否清晰
3. 引言是否与 Results 对齐
4. 最后一段目标是否与研究实际一致
5. 文献是否可能存在弱匹配或不可靠问题

## 输入
- Introduction 初稿
- Validated Results summary
- Topic

## 输出
- Introduction 质控报告
- 修订后的 Introduction

## 高风险信号
- 引言很顺，但换个研究也能用
- 最后一段目标和 Results 对不上
- 文献很多，但都太泛
- 段落像教材，不像论文引言

## 完成标志
- [ ] 引言已与 Results 对齐
- [ ] gap 明确
- [ ] 文献风险点已标记
- [ ] 泛化内容已收紧
- [ ] 引言可进入下一步
