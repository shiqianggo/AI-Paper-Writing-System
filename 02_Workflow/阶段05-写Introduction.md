# 阶段05：写 Introduction

## 目标
写出服务于本研究结果的引言，而不是泛泛的领域综述。

## 这一阶段要做什么
1. 基于研究主题、研究系统和已验证的 Results 写引言
2. 引入必要文献
3. 建立背景 → gap → 本研究目标的逻辑链
4. 输出文中引用和参考文献草稿

## 输入
- Topic
- Study system
- Validated Results summary
- Core concepts
- Journal style preference

## 输出
- Introduction 初稿
- 文中引用
- 参考文献草稿

## 核心要求
- 引言必须和实际 Results 对齐
- 不能写成泛泛的综述
- 要明确：
  - 问题是什么
  - 为什么重要
  - 目前缺什么
  - 本研究解决什么

## 注意事项
- 文献可以由 AI 初步整理，但必须后续核查
- 引言的最后一段要与论文实际目标严格对齐
- 避免过多空泛背景

## 完成标志
- [ ] 引言初稿已生成
- [ ] gap 明确
- [ ] 最后一段目标清楚
- [ ] 文献已初步插入
- [ ] 进入 Introduction 专项质控
