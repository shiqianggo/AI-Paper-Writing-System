# 阶段00：固定研究事实材料

## 目标
在 AI 开始写作前，先固定论文中的“硬事实层”，确保后续生成建立在可靠材料基础上。

## 这一阶段要做什么
1. 整理主图、补图、表格
2. 完成图注
3. 整理 Methods / Materials and Methods
4. 整理 Supplementary materials
5. 单独列出核心结果数字表
6. 明确研究问题与创新点
7. 若已考虑投稿方向，可初步记录目标期刊范围

## 输入
- Figures
- Figure legends
- Methods
- Supplementary
- Key numbers table
- Research question summary
- Novelty summary

## 输出
- 一份完整的研究事实材料包
- 研究信息卡
- 缺失信息清单

## 为什么重要
这一阶段的质量决定了后续 AI 输出的上限。  
如果事实层不固定，AI 很容易在后面：
- 编造数字
- 混淆结果
- 误读方法
- 写出不稳定的章节内容

## 注意事项
- 图注尽量完整，不要只写标题
- Methods 尽量由作者自己主导
- 核心数字最好单独列成表
- 不要让 AI 在这一步替你定义科学事实

## 完成标志
- [ ] 图表齐全
- [ ] 图注齐全
- [ ] Methods 已整理
- [ ] Supplementary 已整理
- [ ] 核心结果数字表已建立
- [ ] 研究问题已明确
- [ ] 创新点已明确
