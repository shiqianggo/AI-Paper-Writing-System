# 阶段10：摘要与标题专项质控

## 目标
检查摘要和标题是否比正文更夸张、更强或更模板化。

## 这一阶段要做什么
重点检查：
1. Abstract 的结论是否都被 Results 支持
2. Title 是否比正文更强
3. Keywords 是否准确反映全文重点
4. 摘要是否过度模板化
5. 是否有比正文更强的因果/机制表述

## 输入
- Title
- Abstract
- Keywords
- Introduction
- Results
- Discussion

## 输出
- 摘要/标题质控报告
- 修订后的 Abstract / Title / Keywords

## 高风险信号
- Title 用 `drive`，正文只是 `associated with`
- 摘要把 Discussion 的推测写成定论
- 关键词太泛或偏离正文重点

## 完成标志
- [ ] 摘要与正文一致
- [ ] 标题强度已校准
- [ ] 关键词与正文一致
- [ ] 摘要模板感已降低
