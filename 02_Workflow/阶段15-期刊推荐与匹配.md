# 阶段15：期刊推荐与匹配

## 目标
根据论文内容推荐候选期刊，并评估匹配度与风险。

## 这一阶段要做什么
1. 基于 Title、Abstract、核心结果做期刊推荐
2. 区分稳妥匹配和冲刺型选择
3. 分析各期刊的匹配理由与潜在短板
4. 为后续定向润色提供依据

## 输入
- Title
- Abstract
- Validated Results summary
- Field
- Journal constraints

## 输出
- 候选期刊列表
- 匹配理由
- 潜在风险
- 针对不同期刊的修改建议

## 注意事项
- 这一步建议放在主体稿稳定后做
- 不宜过早决定，否则容易影响前面写作判断
- 期刊推荐不是最终投稿决定，仍需作者判断

## 完成标志
- [ ] 候选期刊清单已生成
- [ ] 匹配逻辑已清楚
- [ ] 风险点已记录
- [ ] 已确定目标期刊或目标范围
