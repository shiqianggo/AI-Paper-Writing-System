# 阶段12：全文一致性检查

## 目标
确保 Title、Abstract、Introduction、Results、Discussion 和 Keywords 围绕同一主线，术语统一、强度一致。

## 这一阶段要做什么
重点检查：
1. Abstract 中的结论是否被 Results 支持
2. Introduction 提出的问题是否在 Results 和 Discussion 中得到回应
3. Discussion 是否脱离 Results
4. 术语、缩写、概念是否统一
5. 标题是否比正文更强
6. 不同章节是否重复同一结论但措辞不一致
7. Keywords 是否反映真实重点

## 输入
- Title
- Abstract
- Keywords
- Introduction
- Results
- Discussion

## 输出
- 一致性问题报告
- 建议的统一修改方案

## 为什么重要
分章节生成很容易出现：
- 每章都对
- 合起来不完全一致

这一阶段的作用就是“整稿统一”。

## 完成标志
- [ ] 主线统一
- [ ] 术语统一
- [ ] 各章节强度统一
- [ ] 摘要和正文一致
- [ ] Discussion 与 Results 对应
