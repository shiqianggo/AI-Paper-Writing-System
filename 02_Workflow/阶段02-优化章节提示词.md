# 阶段02：优化章节提示词

## 目标
把作者写的“章节任务单”转成结构清晰、约束明确、适合大模型执行的高质量 prompt。

## 这一阶段要做什么
1. 输入原始章节任务说明
2. 让 AI 重构为更专业的 prompt
3. 检查新 prompt 是否包含：
   - 角色设定
   - 输入优先级
   - 段落结构
   - 证据边界
   - 缺失信息规则
   - 输出格式

## 输入
- 原始章节任务单
- 文风要求
- 禁止事项
- 输出语言要求

## 输出
- 章节专用高质量 prompt

## 为什么重要
好的 prompt 不是写得长，而是：
- 约束清楚
- 目标明确
- 证据边界明确
- 容易执行
- 输出稳定

## 注意事项
- 这一步不要直接生成论文正文
- 优先优化 Results prompt
- 记录优化前后差异，方便后续复用
