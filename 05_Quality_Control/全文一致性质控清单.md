# 全文一致性质控清单

## 1. 主线一致性
- [ ] Introduction、Results、Discussion 是否围绕同一研究问题
- [ ] 核心创新点是否前后一致
- [ ] 研究对象和尺度是否表述一致

## 2. 术语一致性
- [ ] 同一对象是否始终使用统一名称
- [ ] 缩写是否定义一致
- [ ] 指标和概念是否前后一致

## 3. 结论强度一致性
- [ ] Results、Discussion、Abstract 的结论强度是否一致
- [ ] 是否有局部段落明显比全文更满
- [ ] 标题是否超出正文支持范围

## 4. 图表与编号
- [ ] 图号、表号、补充材料编号是否一致
- [ ] 正文引用是否准确
- [ ] 是否有漏引或错引

## 5. 信息一致性
- [ ] 关键数字前后一致
- [ ] 分组、年份、地点、样本信息一致
- [ ] 方法名称或分析流程表述一致

## 6. 输出判断
- [ ] 可进入专项核查阶段
- [ ] 需统一术语
- [ ] 需统一主线或创新点表达
