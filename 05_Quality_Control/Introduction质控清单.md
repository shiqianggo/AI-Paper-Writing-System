# Introduction质控清单

## 1. 逻辑链
- [ ] 是否形成“背景—已知—未知—为什么重要—本研究做什么”的链条
- [ ] 研究空白是否清楚
- [ ] 研究问题是否自然引出
- [ ] 研究目标是否明确收束

## 2. 与研究匹配度
- [ ] 引言是否和真实研究主题一致
- [ ] 是否提前写满结论
- [ ] 是否把研究意义拔高过头
- [ ] 是否与后文结果主线一致

## 3. 文献与支撑
- [ ] 关键背景句是否需要引用
- [ ] 未核实引用是否保留 `[Reference Needed]`
- [ ] 是否有泛泛而谈但无支撑的句子
- [ ] 是否过度依赖模板化背景综述

## 4. 风格问题
- [ ] 是否过于空泛
- [ ] 是否像“综述开头”而不像“论文引言”
- [ ] 是否有明显 AI 模板句堆积

## 5. 输出判断
- [ ] 可以进入 Discussion 写作
- [ ] 需要补文献
- [ ] 需要重构逻辑链
