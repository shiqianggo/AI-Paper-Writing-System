# 摘要标题质控清单

## 1. 标题
- [ ] 标题是否忠于证据
- [ ] 是否使用了过强动词
- [ ] 是否把局部研究写成普遍规律
- [ ] 是否准确体现研究对象和主题

## 2. 摘要结构
- [ ] 是否包含背景、目标、方法、结果、结论
- [ ] 结果是否抓住核心发现
- [ ] 是否遗漏关键结果
- [ ] 是否出现正文没有支持的结论

## 3. 结论强度
- [ ] 摘要结论是否强于正文
- [ ] 是否把 suggests 写成 demonstrates
- [ ] 是否把相关模式写成驱动机制
- [ ] 是否有过满、过亮点化表达

## 4. 数字与细节
- [ ] 关键数字是否准确
- [ ] 缺失数字是否保留占位符
- [ ] 是否存在过于笼统但应给数字支撑的句子

## 5. 风格
- [ ] 是否过于光滑、空泛
- [ ] 是否过度营销化
- [ ] 是否符合 SCI 摘要风格

## 6. 输出判断
- [ ] 可进入整稿一致性检查
- [ ] 需降级标题或摘要结论
- [ ] 需补关键结果信息
