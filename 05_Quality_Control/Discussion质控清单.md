# Discussion质控清单

## 1. 证据边界
- [ ] 是否区分了结果、解释和推测
- [ ] 是否有超出结果支持范围的句子
- [ ] 是否把机制写成已证实事实
- [ ] 是否把相关性写成因果

## 2. 研究特异性
- [ ] 是否真正围绕本研究结果展开
- [ ] 是否出现“什么论文都能套”的模板段
- [ ] 是否体现研究对象、尺度、数据类型的特异性
- [ ] 是否把创新点说清楚但不过度拔高

## 3. 文献使用
- [ ] 关键比较句是否需要引用
- [ ] 是否把文献当作本研究证据替代品
- [ ] 是否缺少与前人研究的有效对话
- [ ] 是否有 `[Reference Needed]` 尚未处理

## 4. 结构
- [ ] 开头是否回应主要发现
- [ ] 中段是否围绕关键结果展开解释
- [ ] 结尾是否适度总结而不过满
- [ ] 是否有局限性或边界意识

## 5. 风格问题
- [ ] 是否过于模板化
- [ ] 是否空泛总结过多
- [ ] 是否有明显 AI 味

## 6. 输出判断
- [ ] 可进入摘要写作
- [ ] 需削弱越界表述
- [ ] 需增强研究特异性
