# 文献与引用质控清单

## 1. 引用需求
- [ ] 关键背景句是否都有引用支持
- [ ] 关键比较句是否都有引用支持
- [ ] 机制推测句是否需要引用
- [ ] 是否仍存在 `[Reference Needed]`

## 2. 引用质量
- [ ] 文献是否真实存在
- [ ] 文献是否真的支持当前句子
- [ ] 支持强度是否匹配句子强度
- [ ] 是否把综述当作原始证据使用

## 3. 引用风险
- [ ] 是否有不相关文献硬撑论点
- [ ] 是否有大量堆砌式引用
- [ ] 是否有明显需原始研究却只给综述的情况
- [ ] 是否有年份、作者、期刊信息错误风险

## 4. 参考文献列表
- [ ] 文中引用与参考文献列表是否一致
- [ ] 参考文献格式是否统一
- [ ] 是否符合目标期刊规范

## 5. 输出判断
- [ ] 可进入数字与事实核查
- [ ] 需补文献
- [ ] 需核实高风险引用
