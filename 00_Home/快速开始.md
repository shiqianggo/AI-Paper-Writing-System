

# 快速开始

如果你是第一次进入这个系统，不需要一开始就把所有文件全部读完。  
你只需要先明确两件事：

1. 你现在处于写作的哪个阶段？
2. 你后期想走哪条路线？
   - 轻量非智能体路线
   - 增强智能体路线

本页的目标，就是帮助你快速找到当前最该打开的文件。

---

## 一、先用一句话理解这个系统

这个系统不是一个“让 AI 自动写论文”的工具包，  
而是一套用于 **在研究事实不失真的前提下，系统化使用 AI 完成论文写作、质控与投稿前优化** 的知识库。

它的核心顺序是：

```text
先固定事实
→ 再组织任务
→ 再生成文本
→ 再做质控
→ 最后进入后期投稿准备
```

而后期投稿准备又分成两条路线：

```
路线 A：阶段15-18（轻量非智能体后期路线）路线 B：08_Agent_System（增强智能体后期路线）
```

---

## 二、第一次使用，先看这 3 个文件

如果你完全是第一次使用，建议先按这个顺序打开：

1. [[01_Principles/总原则]]
2. [[00_Home/系统地图]]
3. [[00_Home/使用路径建议]]

这三个文件分别解决：

- **总原则**：告诉你什么能写、什么不能写
- **系统地图**：告诉你整个系统由哪些部分组成
- **使用路径建议**：告诉你不同阶段该走哪条路径

## 入口 E：我已经接近终稿，想使用智能体做增强版投稿前评审与修稿

如果你想做：

- 更完整的内部 review 模拟
- 多 reviewer comments
- 编辑综合判断
- revision roadmap
- 强约束终修

那么进入增强后期路线：

- [[08_Agent_System/README]]
- [[08_Agent_System/Agent总览与调用顺序]]
- [[08_Agent_System/Agent工作流映射]]
- [[08_Agent_System/Agent输入输出接口规范]]

然后按固定顺序调用 5 个 Agent：

1. Field Analysis & Review Architecture Agent
2. Multi-Perspective Review Board Agent
3. Editorial Synthesis & Decision Agent
4. Revision Planning & Comment Consolidation Agent
5. Constrained Manuscript Revision Agent

## 五、先别急着进智能体系统

如果你现在还处于下面这些状态：

- Results 还没稳定
- Introduction 逻辑还没搭好
- Discussion 还是通用模板
- 文献与数字还没做基础核查
- 图表、Methods、Supplementary 还不稳定

那么先不要进入 `08_Agent_System/`。  
此时优先使用：

- [[02_workflow/README]]
- [[03_Prompts/README]]
- [[04_Templates/README]]
- [[05_Quality_Control/README]]

简单说：

> 先把稿子写稳，  
> 再决定是走轻量后期路线，还是增强智能体路线。
