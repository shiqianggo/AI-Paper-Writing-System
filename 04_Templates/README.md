# Templates

> 本文件夹存放 AI-Paper-Writing-System 中可重复使用的标准模板。  
> 与 `03_Prompts/` 不同，这里的内容更偏向“输入骨架、记录格式、检查表模板”。

## 用途
这些模板主要用于：
- 整理输入材料
- 统一章节任务说明
- 固定 prompt 输入格式
- 规范质控记录
- 规范期刊匹配与核查记录

## 使用原则
- Template 是“骨架”，不是最终内容
- 可按项目裁剪，但尽量保持字段稳定
- 缺失信息可暂留占位符，不要脑补
- 建议与 `03_Prompts/` 联动使用

## 推荐顺序
1. 先填 `Template-标准输入包`
2. 再填章节或写作相关模板
3. 再调用对应 Prompt
4. 最后用质控类模板记录问题与修订
